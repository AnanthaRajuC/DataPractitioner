## Installation

```shell
$ sudo apt-get install python3-dev default-libmysqlclient-dev build-essential
$ sudo apt-get install python3-pymysql
$ pip3 install mysqlclient
$ pip3 install mysql-replication
$ pip3 install clickhouse-mysql
```
